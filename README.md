# Student Management System (PHP & MySQL)

A simple web-based student management system implementing CRUD operations.

## Features
- Add student
- Edit student
- Delete student
- View student list

## Tech Stack
- PHP
- MySQL
- HTML
- CSS

## How to Run
1. Install XAMPP
2. Start Apache & MySQL
3. Create database `student_db`
4. Import table `students`
5. Run on `http://localhost/student_management/`
